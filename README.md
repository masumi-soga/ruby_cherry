2.12.5<br>
Rubyの変数にはオブジェクトの値ではなく参照が格納されている
```
a = "Hello"
b = "Hello"
a = b
a.upcase!
puts a,b
a #=> HELLO
b #=> HELLO
```
上記のようにa=bなどとするとオブジェクトの変更が影響する
